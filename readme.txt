Description:
Le jeu du serpent(Snake Game) est un jeu très populaire dans le monde. Simple à comprendre, ce jeu consiste 
pour le joueur a ne amasser le maximum de fruits sans quitter le cadre et sans touher son propre corps qui
croit avec le nombre de fruits amassés.Il se joue seul. 

Caracteristiques:

    -Mode 1 Joueur: Se joue seul
